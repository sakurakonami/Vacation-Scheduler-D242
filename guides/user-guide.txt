2. On Android phone: Open downloaded file

GETTING STARTED:
1. Create account with username and password
2. Login with your credentials
3. Tap "+" to add first vacation
4. Add hotel, dates, and excursions

FEATURES:
- Secure login with password hashing
- Create and manage vacations
- Add excursions with alerts
- Generate vacation reports
- Set notifications for dates

TROUBLESHOOTING:
- Can't login: Check username/password
- App crashes: Reinstall APK
- Notifications not working: Check app permissions
